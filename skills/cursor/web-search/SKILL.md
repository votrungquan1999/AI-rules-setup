# Web Search

Research external information efficiently and critically.

## When to Use

- Current ecosystem changes or breaking updates.
- Non-obvious errors requiring external references.
- Library/tool comparisons.
- Missing local docs for decision-making.

## Workflow

1. Define research question and decision it supports.
2. Craft specific queries with technology/version context.
3. Prefer authoritative sources first:
   - official docs
   - standards/spec pages
   - maintainer issue trackers
4. Cross-check at least 2 sources for critical decisions.
5. Summarize findings and cite links.

## Guardrails

- Avoid broad vague queries.
- Do not trust a single forum snippet blindly.
- Prefer `context7` when the question is strictly about a known library API.
